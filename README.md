# Construction C4 Website

Modern React/Vite landing page for Construction C4.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Recommended: import this GitHub repo into Vercel and deploy automatically.

## GitHub Pages

This repository deploys automatically to GitHub Pages on every push to `main`.
